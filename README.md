Game in Python
=====================

Requirements
------------
```bash
pip install pymunk 
pip install pygame
```

To play
-------
1. Clone the project.
2. Open the src folder.
3. `python main.py`

Zero Gravity
------------
![Alt text](/resources/images/gravity-zero.png?raw=true "angry-birds")
* To activate the "zero gravity mode" press key s.
* To back to normal mode press key n.
